# Employee Management System

##  Description
Employee Management System ek application hai jo company ke employees ka data manage karne ke liye use hota hai. Isme user employee details add, update, delete aur view kar sakta hai.

##  Features
- Add new employee
- Update employee details
- Delete employee record
- View all employees
- Search employee

##  Technologies Used
- Java 
- MySQL 
- HTML, CSS, JavaScript
